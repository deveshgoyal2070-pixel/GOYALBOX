# GOYALBOX

GOYALBOX is a free collection of AI tools and useful online utilities.

This project aims to create a directory of **500+ free tools** including:

- AI tools
- Image tools
- SEO tools
- Developer tools
- Text tools
- Online converters
- Calculators

## Features

- Free tools directory
- Simple search system
- AI and normal tools
- Fast static website
- Hosted using GitHub Pages

## Website

Once deployed with GitHub Pages the website will be available at:

https://USERNAME.github.io/goyalbox

- GitHub Pages
